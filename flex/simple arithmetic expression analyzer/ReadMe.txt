simple.l

ID의 표현
[a-zA-Z_]([a-zA-z_0-9]*)
첫 글자는 숫자가 와선 안되므로 맨 첫 글자는 [a-zA-Z_]로 표기하였다.
두번째 글자부터는 숫자도 허용되며 ID 전체의 길이의 최대 제약은 없으므로 '*'를 붙여 [a-zA-z_0-9]*로 표기하였다.
이 둘을 조합해 [a-zA-Z_]([a-zA-z_0-9]*)로 ID를 나타내도록 하였다.

연산자
기존의 '+', '-'외에도 '*', '/', '(', ')'를 처리할 수 있도록 추가하였다.


simple.y
주어진 production rule을 그대로 표기.
main 시작시 모든 production rule을 출력하도록 제작.

sample
1 : 숫자로 시작하는 ID의 에러처리
2 : ID 인식 확인
3 : ')'만 단독으로 사용
4 : '('만 단독으로 사용
5 : '(', ')'의 인식 확인
6 : 사칙 연산
7 : ()와 사칙연산
